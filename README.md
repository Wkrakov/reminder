# 📚 Студенческая Напоминалка

Веб-приложение для студентов с напоминаниями, категориями и повторами.

## ✨ Функции
- ✅ Добавление напоминаний с датой/времени
- ✅ Категории: учёба, экзамены, личное, прочее
- ✅ Повторяющиеся напоминания (ежедневно/еженедельно)
- ✅ Фильтрация по категориям
- ✅ Отметка выполненных задач
